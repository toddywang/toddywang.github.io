2017-11-21
lodash
Toddy

“Array” 方法

_.chunk(array, [size=1]);
将数组（array）拆分成多个 size 长度的区块，并将这些区块组成一个新数组。 如果array 无法被分割成全部等长的区块，那么最后剩余的元素将组成一个区块。
参数
array (Array): 需要处理的数组
[size=1] (number): 每个数组区块的长度
返回
(Array): 返回一个包含拆分区块的新数组（愚人码头注：相当于一个二维数组）。
